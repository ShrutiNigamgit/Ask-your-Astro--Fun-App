# Ask_Your_Astro

A new Flutter project.

Created an astrology-based mobile application for kids, offering playful graphics and 
amusing responses of yes, no, maybe or don't know to their questions.

The project involved designing and developing a mobile application with an astrology theme 
that was specifically aimed at children. The app allowed children to ask questions, 
and the app provided responses in the form of yes, no, maybe, or don't know,
 with the responses being non-authentic and meant purely for fun. 

The graphics used in the application were designed to be playful and engaging for children,
 adding to the overall fun experience of using the app.
